## 🚀 Getting Started

1. **Connect** the board via USB-C to your computer.
2. **Install** the appropriate board package for:
   - Arduino IDE
   - PlatformIO
   - ESP-IDF / Pico SDK
3. **Flash** a sample project or use one from `/firmware/`
4. **Power** via USB or external battery (if supported)
